# **App Name**: ZenFlow
 
## Core Features:
  
- Dashboard Overview: Presents a clear, card-based view of total income, expenses, remaining budget, and top spending category.
- Spending Categories Management: Allows users to view predefined spending categories with names, assigned colors, monthly budget limits and spent amounts.
- Expense Tracking: Loads and displays expenses from demo JSON data, including expense title, amount, category, date, and payment type.
- Budget Goals Visualization: Visually represents monthly budget goals per category with progress bars and warning states when approaching or exceeding limits.
- Category Spending Breakdown: Displays a pie chart to visualize spending across categories.
- Monthly Expense Trends: Shows a bar chart illustrating monthly expenses.
- Smart Budget Suggestions: AI-powered tool that analyzes spending habits and suggests optimal budget allocations for each category to help users achieve their financial goals. Takes into consideration spending patterns from the expenses JSON.

## Style Guidelines:

- Primary color: Soft, calming blue (#73A5C6) to evoke trust and stability.
- Background color: Light, desaturated blue (#F0F4F7) to create a clean and unobtrusive backdrop.
- Accent color: Warm, muted orange (#C68E73) to highlight key interactive elements and call to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral feel.
- Use simple, outlined icons from a consistent set to represent spending categories and actions.
- Employ a modern, card-based layout to organize information on the dashboard, ensuring a clean and intuitive user experience.
- Incorporate subtle animations (e.g., fade-in, slide-up) for transitions and data loading to enhance engagement without being distracting.
